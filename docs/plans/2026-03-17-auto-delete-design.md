# 自动删除模块设计

## 概述
在群组消息处理链中内联“自动删除”模块，保持优先级：自动封禁 > 自动删除 > 自动回复。触发后立即静默删除，不发送提示消息。

## 数据模型
- KV: `group:{group_id}:auto_delete`
- 字段：
  - 开关项：系统消息、频道马甲、链接消息、超长消息、视频消息、贴纸消息、禁止转发、删除广告贴纸、压缩包、可执行文件、文档、提醒文字、删除@、其他命令、二维码、删除编辑消息、包含会员表情、单独发送会员表情、删除外部回复、删除分享联系人、排除管理员
  - `long_length`: 超长阈值（默认 500）
  - `custom_rules`: 自定义删除规则列表（id、keyword、mode）
  - `ad_sticker_ids`: 广告贴纸 file_unique_id 列表

## 删除触发与优先级
- 普通消息：先入群验证，再自动删除，未命中才自动回复。
- 频道马甲：消息 sender_chat 为 channel 时直接进入自动删除判断。
- 系统消息：根据“系统消息”开关决定是否删除。
- 编辑消息：若“删除编辑消息”开启，编辑后的消息立即删除。
- 管理员排除：开启后管理员消息不触发删除。

## 规则与判定
- 规则匹配支持精确/包含/正则。
- 链接、提及、二维码、外部引用、分享联系人等按消息实体或内容特征判断。
- 触发删除后立即执行，不发送提示。

## 菜单与交互
- 主菜单进入“自动删除”。
- 展示所有开关（两列布局）与“添加自定义规则 / 全部自定义规则”。
- 添加规则：输入关键词 → 选择匹配模式 → 保存。
- 列表支持删除规则。

## 错误处理
- 删除失败（权限不足/消息已不存在）记录日志，不提示用户。
- 正则错误视为不命中。

## 测试要点
1. 链接/超长/视频/贴纸/转发/二维码等逐项触发删除。
2. 自定义规则三种匹配模式均可触发删除。
3. 排除管理员开/关验证。
4. 编辑消息删除。
5. 频道马甲与系统消息删除。
